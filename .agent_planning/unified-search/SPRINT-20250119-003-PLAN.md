## Technical Architecture

### Text Extraction Workflow

```
Data saved:
  SaveRequest(request)
    ↓
  textExtractor.ExtractRequestText(request)
    → returns normalized string
    ↓
  indexRequestFTS(text)
    → INSERT into requests_fts
```

### Search Workflow

```
Frontend: useSearchRequests(query)
  ↓
GET /api/v2/requests/search?q=...
  ↓
searchBuilder.BuildFTSQuery(query)
  → escapes, splits, builds FTS5 expression
  ↓
SearchRequests(ftsQuery)
  → executes FTS query
  ↓
Results with snippets
```
